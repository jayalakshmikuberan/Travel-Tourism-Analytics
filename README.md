# Hotel Booking Analytics Dashboard

## Project Overview
This project analyzes hotel booking data using Power BI.

## Tools Used
- Power BI
- Python
- Pandas
- NumPy

## Dataset
Hotel Booking Dataset

## Features
- KPI Cards
- Booking Trend Analysis
- Cancellation Analysis
- Customer Type Analysis
- Market Segment Analysis
- Country Analysis
- Interactive Slicers

## Dashboard Pages

### Page 1
- Total Bookings
- Total Cancellations
- Average ADR
- Average Lead Time
- Booking Trend
- Hotel Distribution

### Page 2
- Reservation Status
- Market Segment
- Customer Type
- Country Analysis
